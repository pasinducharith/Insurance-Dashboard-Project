# Insurance Claims Dashboard

## Project Overview
This project is an interactive Excel dashboard developed using an insurance dataset containing demographic, medical, and insurance claim information. The dashboard helps analyze how factors such as age, BMI, smoking habits, diabetes, gender, and region affect insurance claim amounts.

The main objective of this project is to transform raw insurance data into meaningful business insights using Microsoft Excel data analysis and visualization tools.

---

## Dataset Description
The dataset includes information related to:

- Age
- Gender
- BMI (Body Mass Index)
- Blood Pressure
- Smoking Status
- Diabetic Condition
- Number of Children
- Region
- Insurance Claim Amount

The dataset is used to identify trends, compare insurance claims among different customer groups, and understand the relationship between medical conditions and healthcare expenses.

---

## Dashboard Features
The dashboard contains:

- Total Claims KPI
- Average Claim KPI
- Total Patients KPI
- Average BMI KPI
- Claims by Region Analysis
- Claims by Gender Analysis
- Smoker vs Non-Smoker Analysis
- Diabetic vs Non-Diabetic Analysis
- Interactive Slicers and Filters
